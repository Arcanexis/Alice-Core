# PPT 技能 (HTML 幻灯片生成器)

## 功能
将结构化的演示内容（如标题、章节、要点）快速渲染为单文件 HTML 幻灯片，输出至 `alice_output/` 目录。幻灯片基于轻量级 CSS 实现，支持基本分页与响应式布局，无需 JavaScript 即可浏览。

## 适用场景
- 需要快速生成演示文稿（如日报、简报、汇报提纲）
- 在无 PowerPoint 环境下分享可视化内容
- 将新闻、数据摘要、任务计划等转为可展示格式

## 输入格式
通过 Python 脚本调用时，需提供以下结构化数据：
- `title`: 演示文稿主标题（字符串）
- `slides`: 幻灯片列表，每项包含：
  - `heading`: 该页标题（可选）
  - `content`: 该页正文（支持 Markdown 或纯文本，将自动转为 HTML）

## 输出
生成文件：`alice_output/{自定义文件名}.html`  
特点：单文件、内联样式、可直接在浏览器打开

## 依赖
- Python 标准库（`os`, `string`, `html`）
- 无外部网络请求，完全本地渲染
